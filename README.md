# Polymarket Feed

TikTok-style vertical scroll for Polymarket prediction markets.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Features

- Vertical swipe navigation (touch, scroll, arrow keys)
- Real-time Polymarket data
- Live probability updates
- Black & white minimal design
- Price charts

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Polymarket API

## Navigation

- **Swipe/Scroll**: Navigate between markets
- **Arrow Keys**: Up/Down
- **Share**: Top-right button

## Data Source

Fetches live markets from Polymarket Gamma API. Displays active, non-closed prediction markets with real-time odds.
